CSS

Selector

nth-child

element:nth-child(an + b) /* n start from 0, but 元素 from 1. 即第一个元素id是1 */

examples:

tr:nth-child(2n+1)
tr:nth-child(odd)
tr:nth-child(even)
span:nth-child(1) /* span的父节点的第一个孩子 */
span:nth-child(-n+3) /* 前三个 */

CSS Div中居中：

horizontal:

     <div style=‘margin: auto’>
     </div>
</div>

a text-aligned parent can align all the text of its children. inline-block is treated as text; so use inline-block, don’t need margin:auto

vertical:

<div style=‘display: table’>
     <div style=‘display: inline-block; vertical-align: middle’>
     </div>
</div>
